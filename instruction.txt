1) Открыть папку с программой
2) Запустить cmd, введя в адресную строку "cmd"
3.1) Для запуска глубинного поиска нужно ввести "WebParser_CPP.exe {filename for read (url)} {filename for write} {lvl} {proxy url (protocol://username:password@host:port)}" 

WebParser_CPP.exe read.txt write.txt 2 http://user:pass@111.111.111.111:5555

3.2) Для запуска обычного парсера нужно ввести "WebParser_CPP.exe {filename for read (words)} {filename for write} {start page} {end page} {lvl} {proxy url (protocol://username:password@host:port)}"

WebParser_CPP.exe read.txt write.txt 0 10 1 http://user:pass@111.111.111.111:5555

4) Нажать "Enter"
